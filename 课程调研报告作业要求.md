# 课程调研报告作业要求

## 一、背景与目的

随着大语言模型（LLM）和智能体（Agent）技术的快速发展，传统的商业智能（BI）正逐步向**Agentic Business Intelligence（ABI）** 演进。ABI 将智能体（能够感知、推理、决策与执行的自主实体）与商业数据分析深度融合，旨在构建能够主动理解业务需求、动态访问异构数据源、自主进行智能决策并协同完成复杂分析任务的下一代数据智能系统。

作为计算机相关专业的高年级本科生，了解 ABI 领域的最新研究进展、关键技术和挑战，对于建立“AI + 数据”的系统观、培养科研素养和工程创新能力具有重要意义。本次调研报告要求你系统梳理 ABI 领域的核心问题——**数据访问**、**智能决策**、**多 Agent 协同**——并综合分析当前主流方法、代表性工作及未来趋势。

## 二、调研内容与具体要求

请围绕以下三个方面（但不限于此）开展文献调研与综合分析：

### 1. 数据访问问题

- **核心问题**：ABI 中智能体如何高效、安全、语义一致地访问多种数据源（结构化数据库、半/非结构化文档、API、数据湖等）？如何实现自然语言到数据查询（如 NL2SQL、Text-to-API）的高准确率转换？如何解决数据权限、隐私保护与实时性之间的矛盾？
- **建议调研方向**：
    - 基于 LLM 的 NL2SQL 方法与改进（如 DIN-SQL、MAC-SQL、CHESS 等）
    - 智能体访问异构数据的工具调用机制（Function Call、Toolformer 等）
    - 数据访问中的 schema 理解、示例增强、少样本学习技术
    - 数据访问中的安全与隐私保护（如差分隐私、访问控制嵌入）

### 2. 智能决策问题

- **核心问题**：在不确定、动态变化的业务环境中，智能体如何利用数据和模型进行合理推理、规划与决策？如何将用户高层意图分解为可执行的分析步骤？如何平衡探索与利用（exploration vs exploitation）并保证决策可解释性？
- **建议调研方向**：
    - 基于强化学习与 LLM 的推理-决策框架（如 ReAct、CoT、ToT、Reflexion）
    - 智能体在数据查询、异常检测、归因分析、预测推荐等场景中的决策方法
    - 决策的不确定性量化与多目标优化
    - 决策过程的可解释性与人机协作机制

### 3. 多 Agent 协同问题

- **核心问题**：在复杂 ABI 任务中（如跨部门数据整合、端到端商业报告生成），如何设计多个智能体之间的通信、协作、竞争与任务分配机制？如何避免协调冲突、提升整体效率与鲁棒性？
- **建议调研方向**：
    - 多 Agent 系统的架构模式（集中式、分散式、层级式、混合式）
    - 基于角色分工的协作策略（如规划 Agent、执行 Agent、批评 Agent）
    - Agent 间的通信协议、共识机制与冲突解决（如辩论、投票、元认知）
    - 多 Agent 协同在 ABI 中的典型应用（如 AutoGen、MetaGPT、CAMEL 等在数据分析任务中的实践）

### 4. （加分项）ABI 系统整体架构与评估

- 如何设计一个端到端的 ABI 系统？如何评估其在真实业务场景中的有效性（准确性、效率、用户满意度、安全性等）？

## 三、调研报告格式与规范

1. **字数要求**：正文（不含标题、摘要、参考文献）不少于 5000 字，不超过 8000 字。
2. **语言**：中文（专业术语可附英文，如 LLM、ABI、NL2SQL）。
3. **结构要求**（需包含以下部分）：
    - 标题
    - 摘要（300 字以内）
    - 关键词（3~5 个）
    - 引言（背景、ABI 定义、研究意义）
    - 现状综述（按数据访问、智能决策、多 Agent 协同分别展开，需描述最新方法、代表性论文及其贡献）
    - 问题与挑战总结（对比分析当前方法的局限性）
    - 未来趋势展望（1~2 个前沿方向，如具身智能、可信 ABI 等）
    - 结论
    - 参考文献（不少于 15 篇，其中近三年（2023-2026）的顶会/顶刊论文不少于 8 篇）
4. **引用格式**：参考 ACM 或 IEEE 格式，并在正文中用 [数字] 标注。

## 四、文献检索与使用建议

- **推荐数据库**：Google Scholar、DBLP、arXiv、CNKI（部分中文文献）、Web of Science
- **推荐会议/期刊**：SIGMOD、VLDB、ICDE、KDD、NeurIPS、ICLR、ACL、EMNLP、TODS、VLDB Journal、TKDE
- **建议关键词**：Agentic Business Intelligence / Agent-based BI / LLM for Data Analytics / Text-to-SQL / Multi-Agent System / Autonomous Decision-Making / Data Access for LLM Agents
