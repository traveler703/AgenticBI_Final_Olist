# 演示脚本（5-8 分钟）

## 0. 开场（30s）
- 项目定位：面向业务人员的自然语言 Agentic BI。
- 技术核心：DeepSeek + LangGraph + MySQL 物理预聚合表 + Streamlit。

## 1. 环境与数据就绪（40s）
- 展示 Docker MySQL 正在运行：`docker ps`
- 展示数据路径：`data/raw/`（9 个 CSV）

## 2. 预聚合层与性能（1min）
- 展示已创建的 `mv_*` 物理预聚合表
- 运行：`python -m utils.benchmark`
- 强调慢查询与快查询的耗时差异（截图放报告）

## 3. 描述性分析（1min）
- 提问：`2017年各州销售额最高的是哪里？`
- 展示返回 SQL（命中 `mv_state_sales`）与图表

## 4. 诊断性分析（1min）
- 提问：`哪些州延迟更严重，可能原因是什么？`
- 展示 `mv_delivery_perf` 指标与决策 Agent 建议

## 5. 预测性分析（1min）
- 提问：`预测未来6周销售额`
- 展示趋势图与置信区间

## 6. 规范性分析（1min）
- 提问：`给出未来3个月优先改进策略`
- 展示三条可执行建议

## 7. 单轮隔离与复合问题（40s）
- 连续提交两个无关问题，展示新问题不会继承上一轮筛选条件
- 提交一个包含多个子问题的复合问题，展示同一轮内必要上下文可正确传递

## 8. 收尾（30s）
- 回扣评分点：物理预聚合优化、多 Agent 协作、四层分析、6+ 图表与可交互界面。
