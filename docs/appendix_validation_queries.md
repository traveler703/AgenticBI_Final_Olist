# 附录：能力验证问题清单

以下问题用于验收系统是否覆盖作业要求中的关键能力：

1. `2017 年 GMV 是多少？按月趋势怎样？`
2. `2017 年各州销售额排名怎样？`
3. `平台整体准时交付率是多少？哪些州延迟最严重？`
4. `哪种支付方式最受欢迎？平均分期数是多少？`
5. `Top 10 品类销售额分别是多少？`
6. `产品重量和运费之间是什么关系？`
7. `根据历史趋势预测未来 6 周销售额。`
8. `为什么某些州的平均配送时长高于全国均值？`
9. `Top 差评风险品类有哪些？`
10. `基于以上分析，给出未来 3 个月三大改进策略。`

## 建议记录项
- 用户问题
- 命中数据源（物理预聚合表 `mv_*` 或 `fact/base`）
- SQL
- 关键图表
- 文字结论
